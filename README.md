# NYC Taxi Data Platform

A production-style **batch ELT data platform** built to process NYC Taxi trip datasets using a containerized data stack.

This project focuses on **deterministic batch processing, explicit data quality signaling, and safe pipeline reruns**, simulating operational patterns commonly used in real-world data platforms.

---

# Tech Stack

- **Orchestration:** Airflow  
- **Processing Engine:** Spark (PySpark)  
- **Warehouse:** PostgreSQL  
- **Transformation:** dbt  
- **BI / Visualization:** Metabase  
- **Infrastructure:** Docker Compose
